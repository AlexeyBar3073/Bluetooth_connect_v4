# Протокол связи — полная спецификация полей

> **Версия прошивки:** 6.1.1
> **Дата:** 2026-04-10
> **Транспорт:** Bluetooth SPP, JSON

---

## Телеметрия (ESP32 → Android)

### Фракционная отправка

| Слой | Период | Когда |
|------|--------|-------|
| **FAST** | 100 мс | Каждый пакет |
| **TRIP** | 500 мс | Каждый 5-й пакет |
| **SERVICE** | 1000 мс | Каждый 10-й пакет |

---

### FAST-слой

| Ключ | Тип | Источник | Описание | Ед. | Диапазон |
|------|-----|----------|----------|-----|----------|
| `spd` | int | EnginePack | Скорость автомобиля | км/ч | 0–220 |
| `rpm` | int | EnginePack | Обороты двигателя | об/мин | 0–6500 |
| `vlt` | float | EnginePack | Напряжение бортсети | В | 12.0–14.5 |
| `eng` | int (0/1) | EnginePack | Двигатель запущен | — | 0, 1 |
| `hl` | int (0/1) | EnginePack | Габаритные огни включены | — | 0, 1 |
| `sel` | string[4] | EnginePack | Позиция селектора АКПП | — | `"P"`, `"R"`, `"N"`, `"D"`… |
| `tcc` | int (0/1) | EnginePack | Блокировка гидротрансформатора | — | 0, 1 |
| `fuel` | float | EnginePack | Остаток топлива в баке | л | 0–`tV` |

**Пример FAST JSON:**
```json
{"tel":{"spd":65,"rpm":2500,"vlt":14.2,"eng":1,"hl":0,"sel":"D","tcc":0,"fuel":35.2}}
```

---

### TRIP-слой = FAST +

| Ключ | Тип | Источник | Описание | Ед. | Диапазон |
|------|-----|----------|----------|-----|----------|
| `odo` | int | TripPack | Одометр (общий пробег) | км | 0–9 000 000 |
| `trip_a` | float | TripPack | Пробег поездки A | км | — |
| `fuel_a` | float | TripPack | Расход за поездку A | л | — |
| `trip_b` | float | TripPack | Пробег поездки B | км | — |
| `fuel_b` | float | TripPack | Расход за поездку B | л | — |
| `trip_cur` | float | TripPack | Текущий пробег поездки | км | — |
| `fuel_cur` | float | TripPack | Текущий расход поездки | л | — |
| `inst` | float | EnginePack | Мгновенный расход | л/100км | — |
| `avg_cur` | float | TripPack | Средний расход за поездку | л/100км | — |
| `avg` | float | TripPack | Средний расход за всё время | л/100км | — |

**Пример TRIP JSON:**
```json
{
  "tel": {
    "spd": 65, "rpm": 2500, "vlt": 14.2, "eng": 1, "hl": 0, "sel": "D", "tcc": 0, "fuel": 35.2,
    "odo": 152345,
    "trip_a": 123.4, "fuel_a": 12.3,
    "trip_b": 5678.0, "fuel_b": 456.7,
    "trip_cur": 45.2, "fuel_cur": 4.5,
    "inst": 8.5, "avg_cur": 9.8, "avg": 10.2
  }
}
```

---

### SERVICE-слой = TRIP + FAST +

| Ключ | Тип | Источник | Описание | Ед. | Диапазон |
|------|-----|----------|----------|-----|----------|
| `t_cool` | float | KlinePack | Температура охлаждающей жидкости | °C | 0–130 |
| `t_atf` | float | KlinePack | Температура масла АКПП | °C | 0–150 |
| `dtc` | string | KlinePack | Коды ошибок ЭБУ через ";" | — | `"P0135;P0141"` |
| `t_int` | float | ClimatePack | Температура в салоне | °C | -40…+60 |
| `t_ext` | float | ClimatePack | Температура за бортом | °C | -50…+60 |
| `tire` | int (0/1) | ClimatePack | Низкое давление в шинах (предупреждение) | — | 0, 1 |
| `wash` | int (0/1) | ClimatePack | Низкий уровень жидкости омывателя (предупреждение) | — | 0, 1 |

**Пример SERVICE JSON:**
```json
{
  "tel": {
    "spd": 65, "rpm": 2500, "vlt": 14.2, "eng": 1, "hl": 0, "sel": "D", "tcc": 0, "fuel": 35.2,
    "odo": 152345,
    "trip_a": 123.4, "fuel_a": 12.3,
    "trip_b": 5678.0, "fuel_b": 456.7,
    "trip_cur": 45.2, "fuel_cur": 4.5,
    "inst": 8.5, "avg_cur": 9.8, "avg": 10.2,
    "t_cool": 87.5, "t_atf": 75.0, "dtc": "P0135;P0141",
    "t_int": 22.0, "t_ext": -5.0, "tire": 0, "wash": 0
  }
}
```

## Команды (Android → ESP32)

| Команда | Параметры | Обработчик | Описание |
|---------|-----------|------------|----------|
| `reset_trip_a` | — | Calculator | Сброс trip_a и fuel_a в 0 |
| `reset_trip_b` | — | Calculator | Сброс trip_b и fuel_b в 0 |
| `reset_avg` | — | Calculator | Сброс среднего расхода |
| `full_tank` | — | Calculator | Заправка: fuel = tV (ёмкость бака) |
| `correct_odo` | `data`: int | Calculator | Установка нового значения одометра |
| `start_telemetry` | — | Protocol | Включить потоковую передачу телеметрии |
| `stop_telemetry` | — | Protocol | Остановить потоковую передачу телеметрии |
| `get_cfg` | — | Protocol | Запрос настроек |
| `set_cfg` | `data`: объект | Protocol | Установка настроек |
| `kl_get_dtc` | — | K-Line | Чтение кодов ошибок ЭБУ |
| `kl_clear_dtc` | — | K-Line | Сброс кодов ошибок ЭБУ |
| `kl_reset_adapt` | — | K-Line | Сброс адаптации АКПП |
| `kl_pump_atf` | — | K-Line | Прокачка жидкости АКПП |

**Примеры команд:**
```json
{"command": "reset_trip_a", "msg_id": 1}
{"command": "correct_odo", "data": 152346, "msg_id": 2}
{"command": "get_cfg", "msg_id": 3}
{"command": "set_cfg", "data": {"tV": 65.0, "iPerf": 250.0, "iCnt": 4, "sSig": 3.0, "kPrt": 1}, "msg_id": 4}
{"command": "kl_get_dtc", "msg_id": 5}
```

## Настройки (get_cfg / set_cfg)

| Ключ | Тип | Описание | По умолчанию |
|------|-----|----------|-------------|
| `tV` | float | Ёмкость топливного бака (л) | 60.0 |
| `iPerf` | float | Производительность одной форсунки (мл/мин) | 250.0 |
| `iCnt` | int | Количество топливных форсунок (шт) | 4 |
| `sSig` | float | Импульсов датчика скорости на 1 метр | 3.0 |
| `kPrt` | int | Протокол K-Line (1=ISO 9141, 2=KWP2000…) | 0 |

**Ответ `get_cfg`:**
```json
{"ack_id": 3, "cfg": {"tV": 60.0, "iPerf": 250.0, "iCnt": 4, "sSig": 3.0, "kPrt": 1, "fw": "6.1.1"}}
```

---

## Квитанции (ESP32 → Android)

На каждую команду ESP32 отправляет подтверждение:

| Ключ | Тип | Описание |
|------|-----|----------|
| `ack_id` | int | `msg_id` из команды Android |

**Пример:**
```json
{"ack_id": 1}
```
